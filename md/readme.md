# Fabric Data Factory Practice Source

This repo is a synthetic multi-format data source used to practice
Microsoft Fabric Data Factory pipeline activities (Copy, Filter, Lookup,
ForEach, If Condition, GetMetadata, etc.) without needing an Azure
subscription or ADLS.

Folders:
- `csv/` — sample sales & employee data
- `json/` — customer and order records
- `md/` — unstructured notes
- `xml/` — inventory data
- `control/` — manifest driving metadata-driven pipelines
